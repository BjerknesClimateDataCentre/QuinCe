This directory contains configuration files for the JUnit tests
Note that quince.properties isn't included - that's built manually
in the junit.uk.ac.exeter.QuinCe.web.system.ResourceManager.LocalTestResourceManager class.

The files are:

column_config.csv:           Configuration of columns and flags used by the QC routines
extract_routines_config.csv: Configuration of the initial data check routines
qc_routines_config.csv:      Configuration of the QC routines
export_config.csv:           Configuration of file export formats
run_types_config.csv:        Configuration of instrument run types
sensor_config.csv            Configuration of required sensor types
